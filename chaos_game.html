Jocul haosului

Jocul haosului (sau jocul haotic) se referă la un algoritm iterativ descris pentru prima dată de matematicianul britanic Michael Barnsley în 1988. Algoritmul generează o figură autosimilară (fractal) prin repetarea unei proceduri extrem de simple, ancorată în colțurile unui poligon cu n laturi.

Algoritmul este următorul:

  1. se alege un punct X la întâmplare în interiorul poligonului
  2. se alege unul dintre colțurile poligonului la întâmplare (C)
  3. se plasează un punct (P) la jumătatea distanței între punctul inițial (X) și colțul ales (C).
  4. P devine noul punct inițial
  5. se alege un alt colț al poligonului (C1)
  6. se plasează un punct P1 la jumătatea distanței dintre P și C1.

După aplicarea algoritmului în cazul unui triunghi se obține o figură cunoscută sub numele de “triunghiul lui Sierpinsky” sau “sita lui Sierpinsky”, a cărui construcție geometrică se obține prin subdivizarea infinită a unui triunghi echilateral în 4 triunghiuri.

Interfața curentă oferă următoarele opțiuni:

Triunghi
Poligonul de referință este un triunghi, alegerea colțurilor este complet aleatoare. Rezultatul simulării este triunghiul lui Sierpinsky
Patrat v1
Poligonul de referință este un pătrat, alegerea colțurilor este aleatoare. Rezultatul simulării este complet aleator, nu fractal.
Patrat v2
Poligonul de referință este un pătrat, alegerea colțurilor este făcută astfel încât colțul următor nu poate fi același cu culțul curent. Rezultatul simulării este fractal.
Patrat v3
Poligonul de referință este un pătrat, alegerea colțurilor este făcută astfel încât colțul următor nu poate fi același cu culțul curent, nici la un colț distanță în sens antiorar. Rezultatul simulării este fractal.
Patrat v4
Poligonul de referință este un pătrat, alegerea colțurilor este făcută astfel încât colțul următor nu poate fi același cu culțul curent, nici la două colțuri distanță. Rezultatul simulării este fractal.
Pentagon
Poligonul de referință este un pentagon, alegerea colțurilor este făcută astfel încât colțul următor nu poate fi vecinul coltului curent, în ambele sensuri. Rezultatul simulării este fractal.
