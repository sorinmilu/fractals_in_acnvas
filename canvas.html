Canvas

Canvas-ul este unul dintre cele câteva elementele standardului HTML5, standard care a adăugat limbajului HTML elementele necesare pentru a permite clienților web să afișeze conținut multimedia. Alte elemente adăugate de acest standard sunt <video>, <audio> fiecare dintre ele reprezentând, ca și canvas, API-uri către funcționalități avansate ale browserelor și chiar ale sistemului de operare.

Canvas este foarte diferit de funcționalitatea clasică a unui browser internet. În varianta clasică, browserul primește o serie de declarații, structurate într-un conținut HTML, pe care le transformă într-o structură de date numită DOM (Document Object Model). DOM-ul este compus din componente ierarhizate arborescent care conțin instrucțiuni cu privire la afișarea conținutului acestora împreună cu conținutul în sine. Motorul de afișare a browserului va aranja pe pagină elementele de conținut din DOM în funcție de instrucțiunile care le însoțesc, asociindu-le evenimente și funcționalități suplimentare dacă sunt solicitate prin Javascript sau CSS.

Canvas-ul funcționează complet diferit, punând la dispoziția utilizatorului o suprafață de desen accesibilă la nivel de pixel. Canvas-ul permite accesarea directă a fiecărui pixel din suprafața interioară, similar cu accesul disponibil într-un software de editare a imaginilor bitmap.

Posibilitatea de a accesa fiecare pixel în parte face posibilă nu doar desenele 2D ci și construcția scenelor 3D. Majoritatea browserelor au implementat legături cu componentele sistemelor de operare care accesează procesoarele grafice (atunci când sunt disponibile) - ex: DirectX in cazul sistemelor Windows - precum și componente software de randare a scenelor 3D pentru dispozitivele lipsite de procesoare grafice.

Astfel, canvasul a devenit o scenă în care se pot “desena” imagini 2D, se pot afișa scene 3D, se pot construi și executa animații complexe, aproape la fel de eficient ca într-o aplicație nativă a sistemului de operare. Aceasta funcționalitate permite dezvoltarea jocurilor 3D care pot fi rulate direct din programele de navigare, aplicații de desen online, chiar aplicații de CAD care sunt accesibile direct din browser, cum este de exemplu clara.io.

Interpretoarele Javascript existente în programele de navigare au fost extinse cu o serie de instrucțiuni care interacționează cu interiorul canvasului (contextul) pentru a desena figuri geometrice. Astfel, canvasul este în acest moment un component cu personalitate dubla: tagul <canvas> reprezintă un component al DOM-ului care se supune acelarași reguli ca orice alt component și este poziționat în pagină de motorul de construcție a paginii în timp ce interiorul acestuia (contextul) este un spațiu de desen care este accesat direct, fără nici o intervenție din partea motorului de construcție.

Exemplu
 function drawSmileyFace(canvasid) {
    var canvas = document.getElementById(canvasid);
    var ctx = canvas.getContext('2d');
    ctx.beginPath();
    ctx.arc(75,75,50,0,Math.PI*2,true);

    ctx.moveTo(110,75);
    ctx.arc(75,75,35,0,Math.PI,false);

    ctx.moveTo(65,65);
    ctx.arc(60,65,5,0,Math.PI*2,true);

    ctx.moveTo(95,65);
    ctx.arc(90,65,5,0,Math.PI*2,true);
    ctx.stroke();
 }
